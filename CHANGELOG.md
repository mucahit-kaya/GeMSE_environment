This pull request updates the CI workflow for Docker image building and makes minor improvements to environment setup scripts. The main focus is to enhance Docker image tagging and deployment automation, while also fixing minor issues in dependency management and build scripts.
Now docker image can be generated properly by github actions for new api changes.
